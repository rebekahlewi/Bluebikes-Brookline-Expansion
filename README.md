# Bluebikes Demand Analysis & Forecasting

Statistical analysis and demand forecasting project examining factors that influence Bluebikes station usage in the Boston / Brookline area.

## Overview
This project applies regression modeling and time-series analysis to understand spatial, demographic, and seasonal drivers of bike-share demand. The analysis includes station-level demand prediction, seasonal decomposition, and interactive visualization.

## Methods
- Feature engineering (distance-based and demographic variables)
- Multiple linear regression
- Monthly dummy variable modeling for seasonality
- Demand forecasting with confidence intervals
- Tableau dashboard for exploratory analysis

## Tools
Excel, Tableau, Statistical Regression
